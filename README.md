# SkyPulse
- In this project i have created a weather application.

## what i have done
- created a weather application using HTML, CSS, and JavaScript
- used OpenWeatherMap API to get the weather data
- used geolocation API to get the user's location
- used local storage to store the user's preferences
- used canvas API to create a dynamic sky background
- used CSS animations to create a smooth and visually appealing interface
- used responsive design to make the application work on all devices
